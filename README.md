# Descripción:
Space Joyride es un runner lateral 2.5D en Unity donde el jugador controla un cohete que avanza automáticamente mientras esquiva láseres, drones y misiles. Se recogen estrellas para aumentar la puntuación y desbloquear cajas sorpresa que activan power-ups temporales. La dificultad aumenta progresivamente.
